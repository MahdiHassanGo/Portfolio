# ✨ My Portfolio

A truly modern, "New Gen" aesthetic web developer portfolio built to showcase projects, skills, and professional experience cleanly and interactively. 

This portfolio heavily leverages **Glassmorphism**, **Infinite Typography Marquees**, and **Fluid Micro-animations** to provide an unforgettable user experience.

---

## 🚀 Features

- **Next-Gen Hero Section**: Features an interactive layout with floating, animated tech-stack capsules surrounding a centerpiece portrait, backed by an enormous, see-through scrolling typography marquee.
- **Glassmorphic Aesthetic**: The UI across the entire site—from the navbar to skill cards to the About Me panel—uses frosted glass (`backdrop-blur`) effects layered over ethereal glowing gradients.
- **Dynamic Projects Showcase**: Automatically alternating layout cards featuring rich project imagery, descriptions, interactive hover-states, and tech stacks powered by `framer-motion`.
- **Responsive Design**: Flawless Mobile-first UI ensuring that interactions, floating layouts, and navigations translate perfectly to smaller screens.
- **Premium Typography**: Pairs Google Fonts **League Gothic** (for bold, striking headers) and **Outfit** (for sleek, highly-readable body text).

---

## 🛠️ Tech Stack & Technologies

**Core:**
- [React (v19)](https://react.dev/)
- [Vite](https://vitejs.dev/)

**Styling & UI:**
- [Tailwind CSS](https://tailwindcss.com/)
- [DaisyUI](https://daisyui.com/)
- Vanilla custom Glassmorphism utilities

**Animations:**
- [Framer Motion](https://www.framer.com/motion/) (Scroll & Orchestrated animations)
- [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) (Entry animations)
- Custom raw CSS `@keyframes` (Marquees, Bouncing floating tags)

---



## 🎨 Sections Overview

- **`Navbar`**: Fixed, frosted glass navigation pane with soft animated underlines and a striking "Download Resume" gradient button.
- **`Home` / Hero**: The visual anchor of the site with complex floating architecture.
- **`About`**: A responsive flex layout pairing your cinematic headshot with a glass-enclosed "About Me" blurb.
- **`Skills`**: A grid layout displaying expertise. Cards glow upon hover and interact with beautifully blurred detail-modals.
- **`Projects`**: High-quality breakdown of projects including features, tech-stacks, and live links.
